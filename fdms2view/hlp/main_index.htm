Fdms2View Help Index

What's this?

Fostex D-80 harddisk recorders stores the eight channels audio streem in ordinary (old) IDE harddisks. But the format how D-80 stores the audio stream is not readable from Windows Explorer as files. This tool helps to access the stored audio stream, and exporting it.

Navigation

Home key: navigate to the first sample.

End key: navigate to the last sample.

Cursor Left key: navigate view area to left one second. (+ALTgr 5sec.)

Cursor Right key: navigate view area to rigt onse second. (+ALTgr 5sec.)

Cursor Up key: Zoom in (+ALTgr *2)

Cursor Dn key: Zoom out (+ALTgr *2)

Right mouse button on (left) channel title area: hide/show a channel.

Ctrl+ Left mouse button: set region start point.

Shift+ Left mouse button: set region end point.

Ctrl+ Scroll down Y mouse roller: Zoom in timeline.

Ctrl+ Scroll up Y mouse roller: Zoom out timeline.

Shift+ Scroll down Y mouse roller: Zoom in amplitude.

Shift+ Scroll up Y mouse roller: Zoom out amplitude.

Scroll down Y mouse roller: Cursor move forward.

Scroll up Y mouse roller: Cursor move backward.

Space: Play and stop.

Left mouse button: set play cursor.

F2: Properties dialog, where input image file or disk can be configured.

F3: Mixer dialog.

F6 or Shift+F6: Next or previous project, if multiple projects are opened.
